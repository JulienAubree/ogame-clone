Architecture : 100% Data-Driven

Game Config (seed)

  • Chaque talent = entree dans talentTree
  • Definit : branche, tier, position, rangs max, prerequis, type d'effet, parametres
  • Les handlers sont references par cle (modify_stat, planet_bonus, timed_buff...)
  • Modifier un talent = modifier le seed, pas le code

Types d'effets (handlers generiques)

  • modify_stat — modifie une stat du flagship
  • global_bonus — bonus permanent au joueur
  • planet_bonus — bonus a la planete stationnee
  • timed_buff — effet temporaire activable (duree + cooldown)
  • unlock — debloque une capacite

Regles de fonctionnement

Progression

  • Libre : investir dans toutes les branches
  • Tier N+1 accessible apres X points dans la branche
  • Certains talents ont des prerequis specifiques (fleches)
  • Respec individuel = 50% du cout investi
  • Full reset = 50 Exilium

Flagship en mission

  • Stats flagship : toujours actives
  • Passifs globaux : toujours actifs
  • Passifs planetaires : cessent au depart
  • Actifs deja lances : persistent jusqu'a expiration
  • Actifs non lances : non activables en mission
